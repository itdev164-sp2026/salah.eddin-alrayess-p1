Requirements Document

Project 1 – Study Tips Website

Project Description
Study Tips is a website that displays study articles for students. The site pulls article content
from a CMS and displays it on a homepage and individual article pages. Users browse study tips,
open articles, and read the full content. The project uses React with Gatsby for page generation
and Contentful as the CMS.

Feature List
Feature 1
Article list page
Feature 2
Article detail page
Feature 3
Content management system integration
Feature 4
Navigation header
Feature 5
Article images

User Stories
Feature 1.
Article list page
User opens the homepage
User sees a list of study articles
User sees article image, title, and category
Feature 2.
Article detail page
User clicks an article title
User navigates to the article page
User reads the full article
Feature 3.
CMS integration
Admin creates an article in Contentful
Admin adds title, category, body text, and image
Website automatically loads the article content
Feature 4.
Navigation header
User sees the site header
User can click the header title to return to the homepage
Feature 5.
Article images
User sees a featured image for each article
User sees the image again on the article page

Tasks
Create Gatsby project
Connect Gatsby to Contentful
Create Contentful content model for articles
Add article entries in Contentful
Create homepage that displays articles
Create dynamic article pages using slugs
Add navigation header
Display article image and content on the article page
